# Parkinson Voice Dataset

## Description
This dataset contains voice recordings used for Parkinson's Disease detection using Machine Learning.

## Dataset Details

- Total Audio Samples: 1134
- Healthy Samples: 574
- Parkinson Samples: 560
- Audio Format: WAV

## Classes

1. Healthy
2. Parkinson's Disease


## Feature Extraction

The following audio features were extracted:

- MFCC (Mel Frequency Cepstral Coefficients)
- Chroma Features
- Mel Spectrogram
- Spectral Contrast
- Tonnetz Features
- Zero Crossing Rate
- RMS Energy

Total Features Extracted: 198
